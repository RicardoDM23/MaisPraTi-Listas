Informações detalhadas das matérias 1. Introdução ao Python e Configuração do Ambiente Objetivos: Familiarizar-se com a linguagem Python e configurar o ambiente de desenvolvimento. Conteúdos: Instalação do Python, configuração de IDEs, sintaxe básica, tipos de dados e operadores, primeiros scripts. 2. Estruturas de Dados e Controle de Fluxo Objetivos: Compreender e utilizar estruturas de dados e controle de fluxo. Conteúdos: Listas, tuplas, conjuntos, dicionários, loops (for, while), condicionais (if, elif, else). 3. Funções e Módulos Objetivos: Dominar a criação e uso de funções e a modularização de código. Conteúdos: Definição de funções, argumentos e parâmetros, escopo de variáveis, importação e criação de módulos, pacotes. 4. Programação Orientada a Objetos Objetivos: Aprender os conceitos e aplicações da POO em Python. Conteúdos: Classes e objetos, atributos e métodos, encapsulamento, herança, polimorfismo, métodos especiais. 5. Manipulação de Arquivos e Exceções Objetivos: Gerenciar arquivos e lidar com exceções no código. Conteúdos: Leitura e escrita de arquivos, modos de abertura de arquivos, tratamento de exceções com try, except, finally. 6. Bibliotecas para Análise de Dados Objetivos: Utilizar bibliotecas para manipulação e análise de dados. Conteúdos: Introdução ao Pandas, criação e manipulação de DataFrames, operações com NumPy, limpeza e transformação de dados. 7. Visualização de Dados Objetivos: Criar visualizações de dados informativas e estéticas. Conteúdos: Uso do Matplotlib para gráficos básicos, personalização de gráficos, criação de gráficos avançados com Seaborn. 8. Desenvolvimento Web com Django Objetivos: Desenvolver aplicações web completas com Django. Conteúdos: Configuração do ambiente Django, estrutura de projetos e aplicativos, views, templates, models, formulários, autenticação de usuários. 9. Introdução a Machine Learning Objetivos: Conhecer os conceitos e práticas de machine learning. Conteúdos: Introdução ao machine learning, tipos de aprendizado (supervisionado, não supervisionado), implementação de modelos com Scikit-learn, avaliação de modelos. 10. Projeto Final Objetivos: Integrar e aplicar os conhecimentos adquiridos ao longo do curso. Conteúdos: Definição do projeto, planejamento e execução, integração de diferentes áreas do conhecimento, apresentação e defesa do projeto.

Introdução ao Python e Configuração do Ambiente

Objetivos:

  • Familiarizar-se com a linguagem Python e sua filosofia.
  • Configurar o ambiente de desenvolvimento para a programação em Python.
  • Escrever e executar os primeiros scripts em Python.

Conteúdos:

  • Instalação do Python: Passo a passo da instalação do Python em diferentes sistemas operacionais.
  • Configuração de IDEs: Introdução às principais IDEs (Integrated Development Environments) como PyCharm, VSCode e Jupyter Notebook, e como configurá-las.
  • Sintaxe Básica: Estrutura básica de um programa em Python, indentação, comentários.
  • Tipos de Dados e Operadores: Inteiros, floats, strings, booleanos, operadores aritméticos, relacionais, lógicos.
  • Primeiros Scripts: Escrever scripts simples para realizar operações básicas e interagir com o usuário.

Estruturas de Dados e Controle de Fluxo

Objetivos:

  • Compreender e utilizar as principais estruturas de dados em Python.
  • Controlar o fluxo de execução dos programas usando estruturas de controle.

Conteúdos:

  • Listas: Criação, acesso, modificação, métodos e operações comuns.
  • Tuplas: Criação, características imutáveis, uso e vantagens.
  • Conjuntos: Criação, operações de conjunto, métodos úteis.
  • Dicionários: Criação, acesso a valores, métodos e iteração.
  • Loops: Uso de loops for e while, compreensão do controle de fluxo com break e continue.
  • Condicionais: Estrutura e uso de if, elif e else para tomadas de decisão.

Funções e Módulos

Objetivos:

  • Dominar a criação e uso de funções para modularizar e reutilizar código.
  • Entender a importância da modularização e como utilizar módulos e pacotes em Python.

Conteúdos:

  • Definição de Funções: Sintaxe, argumentos, valores de retorno.
  • Argumentos e Parâmetros: Diferenciação entre parâmetros e argumentos, argumentos posicionais e nomeados, valores padrão, argumentos arbitrários.
  • Escopo de Variáveis: Variáveis locais e globais, palavra-chave global.
  • Importação de Módulos: Uso de import, from...import, as, módulos padrão da biblioteca Python.
  • Criação de Módulos: Como dividir código em diferentes arquivos e organizá-los em pacotes.

Programação Orientada a Objetos

Objetivos:

  • Aprender os conceitos fundamentais da Programação Orientada a Objetos (POO) e aplicá-los em Python.
  • Desenvolver habilidades para criar e manipular classes e objetos.

Conteúdos:

  • Classes e Objetos: Definição de classes, criação de objetos, construtores.
  • Atributos e Métodos: Atributos de instância e de classe, métodos comuns.
  • Encapsulamento: Princípio do encapsulamento, atributos e métodos privados.
  • Herança: Criação de subclasses, uso de super(), herança simples e múltipla.
  • Polimorfismo: Métodos polimórficos, sobrecarga e sobrescrita de métodos.
  • Métodos Especiais: Métodos __init__, __str__, __repr__, operadores sobrecarregados.

Manipulação de Arquivos e Exceções

Objetivos:

  • Aprender a ler e escrever arquivos de texto e binários.
  • Entender como tratar exceções e erros para tornar os programas mais robustos.

Conteúdos:

  • Leitura e Escrita de Arquivos: Métodos open(), read(), write(), close(), leitura e escrita de arquivos de texto e binários.
  • Modos de Abertura de Arquivos: Modos 'r', 'w', 'a', 'b', 'x', e suas combinações.
  • Tratamento de Exceções: Uso de try, except, finally, tratamento de múltiplas exceções, criação de exceções personalizadas.

Bibliotecas para Análise de Dados

Objetivos:

  • Utilizar bibliotecas essenciais para a manipulação e análise de dados em Python.
  • Capacitar os alunos a trabalhar com grandes volumes de dados de forma eficiente.

Conteúdos:

  • Introdução ao Pandas: Estrutura e manipulação de DataFrames e Series.
  • Criação e Manipulação de DataFrames: Importação de dados, seleção, filtragem, agregação.
  • Operações com NumPy: Criação e manipulação de arrays, operações matemáticas e lógicas.
  • Limpeza e Transformação de Dados: Tratamento de dados faltantes, transformação de tipos, normalização.

Visualização de Dados

Objetivos:

  • Criar visualizações de dados que sejam informativas e esteticamente agradáveis.
  • Aprender a utilizar bibliotecas de visualização para comunicar insights de dados.

Conteúdos:

  • Uso do Matplotlib: Criação de gráficos básicos (linhas, barras, histogramas), personalização de gráficos (títulos, labels, legendas).
  • Gráficos Avançados com Seaborn: Criação de gráficos avançados (heatmaps, pair plots, violin plots), personalização avançada.

Desenvolvimento Web com Django

Objetivos:

  • Desenvolver aplicações web completas utilizando o framework Django.
  • Aprender a configurar e estruturar projetos web em Django.

Conteúdos:

  • Configuração do Ambiente Django: Instalação e configuração do Django, criação de um novo projeto.
  • Estrutura de Projetos e Aplicativos: Organização de projetos, criação e configuração de aplicativos.
  • Views e Templates: Criação de views, renderização de templates, uso de templates dinâmicos.
  • Models e ORM: Definição de models, migrações de banco de dados, uso do ORM (Object-Relational Mapping) do Django.
  • Formulários: Criação e validação de formulários, manipulação de dados de formulários.
  • Autenticação de Usuários: Criação de sistemas de login e registro, gerenciamento de usuários e permissões.

Introdução a Machine Learning

Objetivos:

  • Conhecer os conceitos fundamentais de machine learning.
  • Implementar e avaliar modelos simples de machine learning utilizando Scikit-learn.

Conteúdos:

  • Introdução ao Machine Learning: Conceitos básicos, tipos de aprendizado (supervisionado, não supervisionado).
  • Implementação de Modelos com Scikit-learn: Importação de dados, preparação de dados, seleção de modelos, treinamento e avaliação.
  • Avaliação de Modelos: Métricas de avaliação, validação cruzada, tuning de hiperparâmetros.

Projeto Final

Objetivos:

  • Integrar e aplicar os conhecimentos adquiridos ao longo do curso em um projeto prático e abrangente.
  • Desenvolver um projeto completo que envolva várias áreas do curso, desde a análise de dados até o desenvolvimento web e machine learning.

Conteúdos:

  • Definição do Projeto: Escolha do tema, definição dos objetivos e escopo do projeto.
  • Planejamento e Execução: Planejamento das etapas do projeto, divisão de tarefas, implementação das funcionalidades.
  • Integração de Diferentes Áreas do Conhecimento: Uso de estruturas de dados, manipulação de arquivos, visualização de dados, desenvolvimento web, machine learning.
  • Apresentação e Defesa do Projeto: Preparação de uma apresentação clara e objetiva, defesa do projeto perante instrutores e colegas.
